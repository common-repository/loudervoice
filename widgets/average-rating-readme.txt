A WordPress Widget to show your average review score and review count from LouderVoice.

== Description ==

This plugin gives you an attractive sidebar widget showing your overall review score and count

== Installation ==

Upload the LouderVoice plugin to your site and activate it.

Go to Appearance -> Widget and drag an instance to your sidebar. Enter your LouderVoice API key and LouderVoice CSS URL.
LouderVoice will give you these it you don't already have them.

Further details and a full instruction manual are available from support@loudervoice.com
